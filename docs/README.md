# VSCODE 快捷键

快速复制一行   shift+alt+ 下箭头(上箭头) 或者 ctrl+c 然后 ctrl+v

选定多个相同的单词   ctrl + d

添加多个光标   Ctrl + Alt + 上箭头(下箭头)

全局替换某写单词    ctrl + h 注意选择全部替换即可

快速定位到某一行   ctrl + g

选择某个区块  按住shift + alt 然后拖动鼠标
